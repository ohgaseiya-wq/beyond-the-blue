# Beyond the Blue — AI制作パイプライン

> コンテンツ制作の技術スタック・ワークフロー設計
> ステータス: 設計済み。セットアップは未着手

## 制作パイプライン

```
Claude Code（脚本・構成）
  ↓
Midjourney / Flux（ビジュアル生成）
  ↓
After Effects or Remotion（モーショングラフィックス）
  ↓
ElevenLabs（AIナレーション音声）
  ↓
DaVinci Resolve（最終編集・カラグレ・SE）
  ↓
n8n（自動化・配信）
```

## コンテンツ体裁

| 項目 | 設定 |
|------|------|
| 映像形式 | モーショングラフィックス × AIボイス × ストーリーテリング |
| 音声 | ElevenLabs（ナレーター。葵は喋らない） |
| トーン | 静かで深い（禅・瞑想的）。悟り五分五分 |
| 競合ポジション | 日本版 Pursuit of Wonder × キャラクターIP |

## プラットフォーム展開順序

```
TikTok（仮説検証・短尺60秒）
  ↓
Instagram Reels（反応の良いテーマを深堀り）
  ↓
YouTube（長尺化・SEOメディアと連携）
```

## コンテンツ種別

| 種別 | 尺 | 用途 |
|------|-----|------|
| ショート | 60秒 | TikTok / Reels / Shorts。仮説検証・認知拡大 |
| ミドル | 3〜5分 | Instagram / YouTube。テーマ深堀り |
| ロング | 10〜20分 | YouTube。葵の物語エピソード |
