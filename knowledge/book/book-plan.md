## ライティングルール

### トーン: 悟り五分五分

怒りでもなく悟りでもなく「なんだかなー」。強い言い方をしない。柔らかく、でも深く。

### 具体的な表現ルール

| NG | OK |
|----|-----|
| 「まずい」 | 「こんなもんか」 |
| 「お前」 | 「君」「あんた」 |
| 「なんでだよ」 | 「なんでだろう」 |
| 「どうでもいい」 | 「もう、いいか」 |
| 「失礼なおじいさんだ」 | 「当たっている」 |
| 「誰にも負けていない自信がある」 | 「それなりにあるはず」 |

### Noteパートのルール

- 「Beyond the Blueの教えでは」は使わない → 一般論として語る
- 専門用語は平易に（「観測者効果」→「面白い実験がある」、「扁桃体」→「脳は」）
- 詰め込みすぎない → セクション間に呼吸を。カフェで友達に話す温度感
- 超有料級の情報を入れる（量子力学・哲学・神経科学等の知見を平易に）

### 空の描写

重要なシーンでは空や青の描写を強めに入れる。BTBの「Blue」を物語に溶かす。
夜空 → 無限に広がる深い青の空、宇宙まで届くような描写。
