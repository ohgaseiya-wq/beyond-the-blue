# 葵（Aoi）— キャラクター設計書

> Beyond the Blue の主人公。Blue Mover の象徴。

## 基本情報

| 項目 | 設定 |
|------|------|
| 名前 | 葵（あおい） |
| 性別 | 中性的（androgynous）。男女どちらにも見える |
| 年齢 | 20代前半 |
| 一人称 | 「私」（中性的。「僕」は使わない） |
| 性格 | 静かな決意。不器用。雑草魂。転んでも立ち上がる |

## 外見

- ボロいダークグレーのパーカー
- 色あせたジーンズ
- 汚れたスニーカー
- 身なりの質素さ = 雑草魂の視覚表現

### 最大の特徴: 目

- 通常: 青く光る目（`#6BB8E0`）
- Epiphany時: ブルー → パープルのグラデーションに変化（`#6BB8E0` → `#7B8EC8` → `#9B7BB8`）

## 表現ルール

- **映像では葵は喋らない**。ナレーターが代弁する
- **書籍では一人称独白**（「私」視点）
- Blue Moverの三つの性質（泥臭さ・問い・信頼）を体現するキャラクター

## 師: おじいさん（蕎麦屋の老人）

| 項目 | 設定 |
|------|------|
| 一人称 | 「わし」（仙人感） |
| 性格 | 飄々としている。押し付けない。問いかける |
